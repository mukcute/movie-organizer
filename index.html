Movie and TV Series Organizer

Tool for retrieving organizing movies/TV Series based data from IMDB

Download this project as a .zip file Download this project as a tar.gz file

Welcome to Movie and TV Series Organizer.

This tool helps you organize your movies and TV series based on data fetched from IMDB. Various uses of the tool include: 1. Fetch information about a particular movie. 2. Organize your media library based on year/genre/director. 3. Clean up any unwanted files in the directory.

Usage:

First set up an alias for the command:

$ alias imdbtool="python /path/to/imdbplugin.py"

Some interesting usage examples:

Show all info about file X-Men.First.Class

$ imdbtool -t /path/to/X-Men.First.Class.2011.720p.BrRip.264.YIFY.mp4

Show all info about 1924 version of 'Sherlock'

$ imdbtool -t /path/to/Sherlock.1924.avi

Show best guess for a misspelled name

$ imdbtool -t "Ture git"

Print movie's rating

$ imdbtool -t Cars | sed -n '/^imdbrating/{n;p;}'

Download movie's poster

$  imdbtool -t Cars | wget `sed -n '/^poster/{n;p;}'`
